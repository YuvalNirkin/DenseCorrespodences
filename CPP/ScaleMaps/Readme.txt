To build the mex file in Visual Studio add the following environment variables:
OPENCV_INCLUDE - OpenCV include directory
MATLAB_INCLUDE - MATLAB include directory
EIGEN_INCLUDE - Eigen 3 include directory

For MATLAB 32-bit:
OPENCV_LIB_X86 - OpenCV 32-bit library directory
MATLAB_LIB_X86 - MATLAB 32-bit library directory

For MATLAB 64-bit
OPENCV_LIB_X64 - OpenCV 64-bit library directory
MATLAB_LIB_X64 - MATLAB 64-bit library directory
